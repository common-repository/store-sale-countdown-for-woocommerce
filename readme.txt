== Description ==

Create sale on your shop site and show countdown for sales products from WooCommerce plugin. We can create sale for all days or day wise.  It shows a countdown box on product list and single product page also you can manage many parameters of countdown and sale discount amount from admin section.

== Installation ==

The plugin can be installed like any other.

1. Log into WordPress
1. Select Plugins, then "Add New"
1. Select Upload Plugin
1. Choose the zip file, then Install and activate

Once installed, you will need to create your first survey by going to Store Sale Countdown For WooCommerce.
